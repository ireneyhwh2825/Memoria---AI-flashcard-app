 # Memoria — AI Flashcard App

> Where your notes become knowledge.

An intelligent AI-powered flashcard web app that transforms your study notes into interactive learning sessions instantly.

## Features

- **AI Flashcard Generation** — paste notes or upload PDF/TXT files, generates flashcards instantly
- **3D Flip Cards** — beautiful card flip animation with Easy/Medium/Hard difficulty rating
- **Smart Weak Deck** — automatically tracks difficult cards and resurfaces them for review
- **Quiz Mode** — multiple choice and AI-judged typed answers with instant feedback
- **Session Analytics** — accuracy charts, mastery tracking, and performance insights
- **Daily Streak System** — builds consistent study habits
- **PDF Export** — download your flashcard decks
- **Dark & Light Mode** — elegant theme toggle

## Built With

- Python
- Streamlit
- Groq LLM API (Llama 3.3 70B)
- Plotly
- fpdf2
